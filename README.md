# vue-virtual-components

## Overview
随着前端功能的日益负责和数据量的增加，部分UI组件库不支持数据量过大的功能。
这个库主要是开发这些常用的虚拟组件。组件为了和UI组件库风格样式一致，可能会使用组件库中的组件。
建议这类组件和组件库配合使用。
